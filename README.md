# Beer Luga PWA Starter

This is a React + Vite starter for Beer Luga.

## Features

- Leaflet/OpenStreetMap map
- User GPS location
- Bar markers from `public/data/bars.json`
- Active happy-hour logic
- Bar detail card
- Directions click tracking placeholder
- PWA manifest starter

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL Vite gives you.

## Deploy

Upload this repo to GitHub, then connect it to Cloudflare Pages or Vercel.

Build command:

```bash
npm run build
```

Output folder:

```bash
dist
```
